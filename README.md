# A Library for Algorithmic Game Theory in Ssreflect/Coq

# Prerequisites

* `coq` `>= 8.7`
* `coq-mathcomp-algebra` `1.9.0`

# Build

Just type

```bash
make && make install
```

in the top-level project directory.
